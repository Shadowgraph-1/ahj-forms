# AHJ Popovers (vanilla JS)

Мини-репозиторий с реализацией Popover'а на чистом JS.

Сборка через Webpack, тесты — Jest (jsdom). CI: GitHub Actions, публикация на GitHub Pages.

Badge сборки (пример, замените <OWNER> и <REPO> на свои):

![Build Status](https://github.com/<OWNER>/<REPO>/actions/workflows/deploy.yml/badge.svg)

GH Pages: https://<OWNER>.github.io/<REPO>/ (замените на ссылки на ваш репозиторий)

Как запустить локально:

```
npm install
npm run start
```

Запустить тесты:

```
npm ci
npm test
```

Инструкция преподавателю: пришлите ссылку на ваш репозиторий с настроенным GitHub Actions и опубликованной страницей.
